= AskVAApi

This API is a facade (technically, it's middleware) for the APIs that were developed on the Dynamics / PowerApps platform on the government Azure cloud. 

== Installation
Ensure the following line is in the root project's Gemfile:

  gem 'askvaapi', path: 'modules/askvaapi'
